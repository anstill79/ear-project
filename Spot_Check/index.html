How to use

Place your phone at ear level in the sound field, with the mic facing the speaker.

1. Set the Frequency, Pres. Level, and Speaker to match the audiometer.

2. Start the tone on the audiometer, then tap Start Monitoring.

3. Wait for the reading to settle, then tap Save Now. Use Save in 3 s if you need a moment to step away before the reading is captured.

4. Change to the next frequency on both the audiometer and this app, and repeat.

Results are saved to the table and grouped by booth and date. Tap the Peak value at any time to reset it.
