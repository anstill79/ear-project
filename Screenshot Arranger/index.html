How to use Screenshot Arranger

Overview

Screenshot Arranger helps you place screenshots of test results for the right and left ears in a grid layout. You can drag and drop images into the grid, and each image can be labeled and edited.

Inserting images

There are a few methods to add images to the grid:

  • Click the "Add Image" button to select an image from your device.
  • Highlight a cell then paste from the clipboard.
  • Highlight a cell then drag and drop an image directly into the grid.
  • Double-click on a cell to add an image to the grid via a file picker.
