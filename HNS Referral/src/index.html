        Admin Instructions

        Welcome to the Admin Panel. You can create, edit, and delete the options that appear in the drop-down selection lists, as well as the guidance result that appears when the user chooses any 3 options. Clicking inside an option will make it active for editing (you can also click the Select button next to it). Each selection section also has a button that can be used to create a brand new option. Delete buttons can be found to the right of each item. When you have selected one option from each section the Guidance section will be ready for your text. Write the Guidance text then click Save and the data will become available to all users who visit the site after that.
